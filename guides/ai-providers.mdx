## Anthropic Claude

Claude Sonnet 4 provides the best code completion quality for Polaris.

<Steps>
  <Step title="Create an Anthropic account">
    1. Visit [console.anthropic.com](https://console.anthropic.com)
    2. Sign up for an account
    3. Add billing information (pay-as-you-go)
  </Step>

  <Step title="Generate an API key">
    1. Go to **API Keys** in the console
    2. Click **Create Key**
    3. Copy your API key (starts with `sk-ant-`)
  </Step>

  <Step title="Add to environment variables">
    ```bash
    ANTHROPIC_API_KEY=sk-ant-api03-...
    ```
  </Step>

  <Step title="Verify installation">
    The required package is already in `package.json`:

    ```json
    {
      "dependencies": {
        "@ai-sdk/anthropic": "^3.0.1"
      }
    }
    ```
  </Step>
</Steps>

### Models used

Polaris uses different Claude models for different features:

| Feature | Model | Why |
|---------|-------|-----|
| Suggestions | Claude 3.7 Sonnet | Fast, high-quality completions |
| Quick Edit | Claude 3.7 Sonnet | Accurate code transformations |
| Conversations | Claude Opus 4 | Complex reasoning and planning |

### Configuration

The AI provider is configured in API routes:

```typescript
import { anthropic } from '@ai-sdk/anthropic';
import { generateText } from 'ai';

const model = anthropic('claude-3-7-sonnet-20250219');

const { text } = await generateText({
  model,
  messages: [...],
  temperature: 0.3,
});
```

## Google Gemini

Gemini 2.0 Flash is a free alternative for development and testing.

<Steps>
  <Step title="Create a Google Cloud account">
    Visit [aistudio.google.com](https://aistudio.google.com) and sign in with your Google account.
  </Step>

  <Step title="Generate an API key">
    1. Click **Get API key** in AI Studio
    2. Create a new API key
    3. Copy the key
  </Step>

  <Step title="Add to environment variables">
    ```bash
    GOOGLE_GENERATIVE_AI_API_KEY=AIza...
    ```
  </Step>

  <Step title="Verify installation">
    The required package is already in `package.json`:

    ```json
    {
      "dependencies": {
        "@ai-sdk/google": "^3.0.1"
      }
    }
    ```
  </Step>
</Steps>

### Model configuration

Gemini uses a single model for all features:

```typescript
import { google } from '@ai-sdk/google';
import { generateText } from 'ai';

const model = google('gemini-2.0-flash-exp');

const { text } = await generateText({
  model,
  messages: [...],
  temperature: 0.3,
});
```

## Choosing a provider

<Tabs>
  <Tab title="Claude (Recommended)">
    **Best for:**
    - Production use
    - Highest quality code suggestions
    - Complex refactoring tasks
    - Multi-step reasoning

    **Pricing:**
    - Pay-as-you-go based on tokens
    - ~$0.003 per 1K input tokens (Sonnet)
    - ~$0.015 per 1K output tokens (Sonnet)
  </Tab>

  <Tab title="Gemini (Free tier)">
    **Best for:**
    - Development and testing
    - Learning how Polaris works
    - Low-volume usage

    **Pricing:**
    - Free tier: 1,500 requests/day
    - Rate limit: 15 RPM
  </Tab>
</Tabs>

## Switching providers

You can switch between providers without code changes:

1. Remove the current provider's API key from `.env.local`
2. Add the new provider's API key
3. Restart your development server

Polaris automatically detects which provider is configured.

## Cost optimization

<Tip>
**Reduce costs** by adjusting debounce timing for suggestions:

Edit `src/features/editor/extensions/suggestion/index.ts` and increase the debounce delay from 300ms to 1000ms.
</Tip>

<Warning>
Never commit API keys to version control. Always use environment variables and add `.env.local` to `.gitignore`.
</Warning>

## Rate limits

### Claude
- Tier 1: 50 RPM, 40,000 TPM
- Tier 2: 1,000 RPM, 80,000 TPM
- Tier 3: 2,000 RPM, 160,000 TPM

### Gemini
- Free tier: 15 RPM, 1,500 requests/day
- Paid tier: 1,000 RPM, 4M tokens/min

## Troubleshooting

<AccordionGroup>
  <Accordion title="API key not working">
    Verify your key is correct and has billing enabled (for Claude) or is within rate limits (for Gemini).
  </Accordion>

  <Accordion title="Suggestions not appearing">
    Check your browser console for errors. Ensure your API key is set and the provider is reachable.
  </Accordion>

  <Accordion title="Rate limit exceeded">
    For Gemini free tier, you've hit the daily limit. Wait 24 hours or upgrade to a paid plan.
  </Accordion>
</AccordionGroup>
